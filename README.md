# 🚨 AI Crime Visualizer
**Final Year Project | Computer Science & Information Security**

Welcome to the official repository for the **AI Crime Visualizer 2.0**. This project leverages Machine Learning to analyze historical crime data and predict potential hotspots for proactive law enforcement.

### 🌐 Live Workbase
You can access the interactive AI Scanner and Secure Portal here:
👉 [**Launch AI Visualizer Web-App**](https://mumiz-ops.github.io/ai-crime-visualizer/)

## 🛠️ Project Core
* **AI Model:** Random Forest Regression for predictive analytics.
* **Security:** Implemented secure login simulation and SHA-256 data conceptualization.
* **Visualization:** Geospatial mapping of crime density.
