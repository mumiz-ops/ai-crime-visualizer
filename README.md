# 🚨 AI Crime Visualizer
**Final Year Project | Computer Science**

### 📖 Project Overview
A data-driven application designed to visualize crime patterns using Machine Learning. This project helps law enforcement and researchers identify "hotspots" and trends.

### 🛠️ Technical Stack
* **Language:** Python
* **AI Models:** (e.g., Random Forest or K-Means Clustering)
* **Visualization:** (e.g., Matplotlib, Folium, or PowerBI)
* **Security:** Implemented AES encryption for sensitive data handling.

### 🚀 Key Features
1. **Trend Prediction:** Uses historical data to forecast future crime peaks.
2. **Interactive Map:** Geospatial visualization of reported incidents.
3. **Data Security:** Secure login and DMZ-inspired architecture for the database.

### Algorithm Choice
I used **Random Forest Regression** because it handles non-linear crime data better than simple linear models.

### Security Implementation
Following my **Information Security** studies, I implemented:
* **Input Validation:** To prevent SQL injection in the crime database.
* **Encryption:** Sensitive victim data is hashed using SHA-256.
